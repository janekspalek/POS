# Další příkazy

### Zobrazení stavu všech portů + přiřazených IP adres
```
show ip interface brief
```

### Zobrazení směrovací tabulky
```
show ip route
```

### Výpis sousedních zařízení
```
show cdp neighbors
```

### Výpis cesty k zařízení na adrese
```
traceroute 10.0.0.1

traceroute google.com
```
